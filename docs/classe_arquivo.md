#### Arquivo
A classe Arquivo é definida em `src/Arquivo.java` e ela possui o unico propósito de Ler um arquivo OliLang

##### Atributos
`(construtor) String caminho` - ele é definido como o **Argumento de Caminho** (revise o arquivo ***arguments.md***) 

##### Métodos
`Ler()`- Este método inicia a leitura do arquivo, não exige nenhum parâmetro em sua requisição
